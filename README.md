#  Agriconnect

Agriconnect est une application en cours de développement pour faciliter la gestion des activités agricoles.

##  Objectif
Permettre aux agriculteurs de :
- gérer leurs produits
- suivre les achats et ventes
- recevoir des notifications importantes

##  Technologies utilisées
- Python (Django)
- GitHub

## ✨ Fonctionnalités
- 🔐 Authentification (Login / Register)
- 🛒 Gestion des achats
- 💰 Gestion des ventes
- 🔔 Notifications
- 📊 Gestion des données agricoles (en cours)
